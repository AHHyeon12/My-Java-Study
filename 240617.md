

#### 용어정리

```
-  캡슐화 - 필드와 관련된 필요한 정보 메소드를 한 곳에 정리
-  정보은닉 - 접근을 제한하여 외부로부터 정보를 보호함(public / private 차이점)

-  인스턴스의 참조를 통해 호출 - instance method  
-  전달하는 파라미터 값에 따라 호출 - static method 
```
