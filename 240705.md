
#### - nullFirst

null값을 정렬 결과의 첫번째로 나올 수 있게함
{ null, null, 1, 2, 3, 4}
#### - nullLast

null값을 정렬 결과의 마지막으로 나올 수 있게함
{ 1, 2, 3, 4,  null, null}
